- `ai-guide.html` - responsible AI guide plan, approved knowledge, handoff rules and connector options.
- `styles.css` - brand styling and responsive layout.
- `script.js` - small mobile menu interaction.
- `assets/` - logo and concept imagery.

## GitHub Pages

This site can be published from a GitHub repository using GitHub Pages.

Suggested early setup:

1. Create a repository, for example `SBT_GitHubSite` or `strange-but-true`.
2. Add these files to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Set source to the main branch and root folder.
5. Save, then wait for GitHub to provide the public Pages URL.

## Before Public Launch

- Add real contact details.
- Replace or clearly label concept/mockup images where needed.
- Decide whether catalogue sales will use Bandcamp, Gumroad, Ko-fi, Payhip, website checkout or manual delivery.
- Add real free samples and paid download descriptions.
- Add ABN/business/insurance details when ready.
